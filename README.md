# Personal Portfolio Website

This is my personal portfolio website developed for the DevOps module assignment.

The website includes my personal information, education, technical skills, projects, contact details, a profile photo, light/dark theme switching, scroll animations, skill progress animations, and a downloadable CV.

- Responsive portfolio website
- Modern dark and light theme design
- Profile photo section
- Smooth scroll animations
- Animated skill progress bars
- Project cards
- Contact section

## CI/CD Pipeline

This project uses GitHub Actions to automatically deploy the website to GitHub Pages.

The workflow file is located at:

.github/workflows/deploy.yml

Whenever changes are pushed to the main branch, GitHub Actions runs the deployment workflow and publishes the latest version of the website.
